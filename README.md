# ERP-IEPI

ERP educacional (Next.js) com módulos acadêmico, financeiro, pedagógico e integrações de pagamento.

## Executar localmente

```bash
npm install
npm run dev
```

Aplicação em `http://localhost:3000`.

## Scripts principais

- `npm run dev` — ambiente local
- `npm run build` — build de produção
- `npm run start` — executar build
- `npm run lint` — lint
- `npm run test` — testes
- `npm run test:coverage` — cobertura
- `npm run test:payment:e2e` — fluxo E2E Cielo
